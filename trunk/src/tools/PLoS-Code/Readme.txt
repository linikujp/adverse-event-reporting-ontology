This folder contains the code and datasets allowing to reproduce results in the paper "The logic of surveillance guidelines: An analysis of vaccine adverse event reports from an ontological perspective." by Courtot et al. and submitted for publication to PLoS Medicine.

The folder CosineSimilarity contains code and datasets to perform the cosine similarity computations and draw the corresponding ROC curves and associated statistics.

The folder VAERSMedDRAContingencyTables contains code and datasets to build the contingency tables as described in the manuscript.

The folder ROCcurve contains the R code and input data files to compute the ROC curves statistics.

More details are in the Readme.txt in each folder.


Upon acceptance, these folders will be uploaded on the public website http://purl.obolibrary.org/obo/aero
